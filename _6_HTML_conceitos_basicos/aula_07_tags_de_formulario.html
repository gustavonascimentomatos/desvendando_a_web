Tags de formulário

<form>: Define um formulário para ser preenchido pelo usuário.
<input>: Define um campo de entrada para o usuário inserir informações.
<select>: Define uma lista suspensa para o usuário selecionar uma opção.
<option>: Define uma opção em uma lista suspensa.
<textarea>: Define uma área de texto para o usuário inserir informações.
<button>: Define um botão para o usuário clicar
