As tags <h1> a <h6>: Define os títulos da página, sendo:
<h1>: O título mais importante
<h6>: O título menos importante

<p>: Define um parágrafo de texto.
<a>: Define um linl para outra página ou recurso.
<em>: Define ênfase em um trecho de texto, normalmente exibido em itálico.
<strong>: Define forte ênfase em um trecho de texto, normalmente exibido em negrito.
<br>: Insere quebra de linha simples.
<span>: Define uma pequena seção de texto dentro de um elemento maior.

<ul>: Define uma lista não ordenada.
<ol>: Define uma lista ordenada.
<li>: Define um item dentro da lista.
