Modulo 02 - Aprendendo lógica de programação - [Javascript]
    Code Runner => Extensão para executar o terminal
    console.log('Teste!')
    var => Antigamente valido para valores mutáveis e imutáveis
    const => Variável, constante que não terá seu valor alterado
    let => Variável, que pode variar os valores armazenados
    '=' => Sinal de atribuição
    '==' => Sinal de comparação independente do tipo
    '===' => Sinal de comparação de valor e tipo
    '[]' => Define a utilização de Array
    '.toFixed()' => Determina quantas casas decimais aparecerão na casa decimal
    ".map" => Utilizado para realizar a conversão de objeto para outro objeto
    ".filter" => Tem que ser devolvido um 'true' ou 'false', Caso o .filter seja indefinido temos que utilizar '!!' para fazer com que a o que for indefinido passe a ser falso.
    ".sort" => Altera direto o objeto que esta sendo ordenado
    ".some" => Valida dentro de uma lista se pelo menos um dos elementos correponde a condição
    ".every" => Valida dentro de uma lista se todos os elementos correspondem a condição
    "Object.defineProperties" => Possível declarar várias propriedades
    "Object.defineProperty" => Possível declarar apenas uma propriedade
